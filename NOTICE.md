# Notice

This project is unofficial and is not affiliated with Siscomex, Portal Único, Receita Federal, or the Brazilian government.

The official source used by the downloader is Siscomex/Classif through Portal Único:

```text
https://portalunico.siscomex.gov.br/classif/api/publico/nomenclatura/download/json
```

This repository does not redistribute the full official NCM dataset. Generated files reflect public official data at the time they are downloaded by the user.

NCM classification has fiscal and legal consequences. This project provides tooling to download, normalize, and search the official table; it does not classify products and does not replace validation by qualified tax, accounting, or customs professionals.

## Aviso

Este projeto não oficial não é afiliado ao Siscomex, Portal Único, Receita Federal ou governo brasileiro.

A fonte oficial usada pelo downloader é o Siscomex/Classif via Portal Único:

```text
https://portalunico.siscomex.gov.br/classif/api/publico/nomenclatura/download/json
```

Este repositório não redistribui a base oficial completa de NCM. Os arquivos gerados refletem dados públicos oficiais no momento em que são baixados pelo usuário.

A classificação NCM tem consequências fiscais e legais. Este projeto fornece ferramentas para baixar, normalizar e pesquisar a tabela oficial; ele não classifica produtos e não substitui validação por profissionais qualificados da área fiscal, contábil ou aduaneira.
